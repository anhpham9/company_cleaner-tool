# Company Cleaner

企業リストを自動で整理・クリーニングするツールです。

---

## 🚀 概要

Company Cleaner は、Excelの企業リストを自動で整理・正規化するツールです。

主な機能：

- 重複データの削除
- 会社名の正規化
- ブラックリストによる除外
- Excelフォーマットの保持
